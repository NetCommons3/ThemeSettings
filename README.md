## テーマ構造

<pre>
 |- ThemeName/ (1) テーマデータの入っているフォルダ
 |		|- webroot/ (2)公開領域
 |		|	|-css/  (3) css格納フォルダ
 |		|	|	|- bootstrap.min.css (4)
 |		|	|	|- style.css         (5)
 |		|	|-img/ (6) 画像格納フォルダ
 |		|	|-fonts/ (7) fontデータ格納フォルダ
 |		|	|-screenshot.png (8) スクリーンショット画像
 |-theme.json (9)設定ファイル
</pre>
